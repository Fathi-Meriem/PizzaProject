# üçï Pizza Order App - C# Windows Forms

This is a simple Windows Forms application built with **C#** that allows users to customize their pizza order dynamically.

## Features

- Choose pizza size (small, medium, large)
- Select crust type (thin, thick)
- Pick multiple toppings from a list
- Select where to eat (eat-in, takeout)
- View a summary of all choices with dynamically updated prices
- User-friendly interface with real-time price calculations based on selections

## How It Works

The app updates the order summary and total price instantly as the user makes selections. This dynamic behavior helps users see the effect of their choices without needing to refresh or submit separately.

## Technologies Used

- C#
- Windows Forms (WinForms)
- Visual Studio

## How to Run

1. Open the solution (`.sln` file) in Visual Studio  
2. Build and run the project (`F5` or `Ctrl + F5`)
